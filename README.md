# BUY MY SHOES 
 My front end repository for my shoe selling app.

## APPROACH TAKEN
This app front end was coded with React processing the data from the Ruby on Rail API. 

## TECHNOLOGIES USED
- Javascript
- React
- CSS
- Milligram
- HTML
